Intro - JavaScript란?

사용자로부터 특정 이벤트나 입력 값을 받아 동적인 처리를 목적으로 고안된 객체 기반의 스크립트 프로그래밍 언어. 예를 들면, 마우스를 HTML 문서의 특정 이미지나 텍스트 위로 가져갔을 때 이미지 크기가 커지거나 텍스트의 색이 바뀌는 기능이다. 또 검색창 입력 박스에 ‘자’를 입력하면 현재 기준으로 사용자 검색이 많은 ‘자’로 시작하는 검색어 목록을 보여주는 기능도 자바스크립트로 구현한 것이다.

html은 1+1을 그대로 1+1이라고 출력하고, JavaScript는 1+1을 2라고 출력한다.

1+1

사용자가 클릭할 수 있는 버튼을 하나 만들어보자. input 태그를 쓴다.
누르면 알림!
버튼을 클릭하면 알림이 뜨도록 했다. onclick이라는 속성은 이벤트(클릭)가 일어났을때 실행하기를 기다리는 것이다.
입력값이 바뀔때마다 알림!
키가 눌릴때마다 알림!
텍스트를 입력할 수 있는 입력창을 만들었다. 입력한 텍스트가 바뀌거나, 키가 눌렸다면 알림이 뜨도록 했다. 이벤트는 이처럼 사용자가 "어떤 행동"을 했을 때, 즉 사건을 정의할 수 있다.
콘솔
'검사'를 누르면 콘솔이라는 탭이 있다. 혹은 element탭에서 esc를 누르면 나타난다. 여기에 자바스크립트 코드를 입력함으로써 내가 지금 보고 있는 특정 웹페이지에서 글자 수를 세거나, 댓글을 추첨하거나 하는 등의 일을 할 수 있다. alert();안에 수를 넣으면 알림으로 뜨고, 그냥 1+1만 적으면 2가 콘솔창에 출력된다.

데이터 타입, 즉 자료형에는 숫자와 문자가 있다. 숫자에는 +,-,*,/등의 산술연산자가 있다. 그리고 문자가 있다. 문자는 항상 "큰따옴표"안에 들어간다. ''안에 문자를 넣고 .length라고 입력하면 글자수를 알려준다. 문자열string이 있고, 이 뒤에 length와 같은 특정한 태그를 붙임으로써 문자와 관련된 기능을 수행할 수 있다. 이러한 자료형에 따라 결과값이 달라진다. 즉, "1"+"1"이라고 입력하면 2가 아닌 11이 출력된다는 것이다.
변수와 대입연산자
변수는 대입연산자를 통해 값이 바뀔 수 있는 수다. =는 오른쪽항의 변수를 왼쪽항에 대입한다는 뜻의 대입연산자다. 1과 같은 수는 항상 값이 바뀌지 않는 상수다. 변수를 언제 사용하는가? name = 'egoing';을 입력하고, 실행을 지연시킬때는 엔터가 아닌 shift+엔터를 친다. 그리고 이름이 들어갈 자리에 "+name+"이라고 적어주면, 텍스트가 출력될때 name 자리에는 언제나 변수값이 들어가게 된다.
야간모드/주간모드 버튼을 만들어보자. 지금까지 배운 js문법을 가지고 제어할 태그를 선택하는 법을 공부할 것이다.


css에서 공부했던 스타일태그. 이것들을 js안에서 표현할 때에는 문법이 조금 달라진다. background-color였던 것이 backgroundColor가 된다. document.queryselector('')안에 선택자를 넣어야하는데, 주의해야할 것은 만약 body에 id나 class를 붙여놓았다면 그 값을 넣어야한다는 것이다. 하지만 이 버튼에서는 그냥 body를 넣으면 된다. 버튼 만들기, 버튼 이름 넣기, 이벤트와 그에 따른 css결과값 출력까지를 알아보았다.
